# 练习题2.17

### 理解题目

给出4位向量中每个位的2的非零次幂，并进行相加。

### 拟定方案

先转换成二进制。

然后根据位向量的。



### 0x5

```
[0101]

无符号:4 + 2 + 1 = 7
补码  :4 + 2 + 1 = 7
```



### 0x5

```
[1000]
无符号: 8
补码  : -8
```





### 0xD

```
[1101]

无符号: 8 + 4 + 1 = 13
补码  : -8 + 4 + 1 = -3
```





### 0xF

```
[1111]
无符号: 15
补码  : -1
```
